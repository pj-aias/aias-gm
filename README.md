# aias-gm
Group Manager of AIAS project.

## Installation

### 1. clone

```sh
git clone https://github.com/pj-aias/aias-gm
cd aias-gm
git checkout develop
```

### 2. set .env

```sh
cp .env.sample .env
vim .env # write
```

### 3. crate db

```sh
touch aias.db
```

### 4. run

```sh
docker-compose up 
# cargo run
```
